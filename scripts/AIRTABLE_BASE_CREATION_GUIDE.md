# 🎯 COMPLETE AIRTABLE BASE CREATION GUIDE

### **Step 1: Create New Airtable Base**

1. **Go to Airtable**
   - Visit [https://airtable.com](https://airtable.com)
   - Sign in to your account

2. **Create New Base**
   - Click **"+ Add a base"** or **"New base"**
   - Choose **"Start from scratch"**
   - Name your base: **"Rensto Business Management"**
   - Click **"Create base"**

### **Step 5: Get Your Base and Table IDs**

1. **Get Base ID**
   - Go to [https://airtable.com/api](https://airtable.com/api)
   - Find your base: **"Rensto Business Management"**
   - Copy the **Base ID** (starts with `app`)

2. **Get Table IDs**
   - Click on your base in the API documentation
   - Copy the **Table ID** for each table:
     - Business References (starts with `tbl`)
     - Customer Management (starts with `tbl`)
     - Project Tracking (starts with `tbl`)

3. **Get API Key**
   - Go to [https://airtable.com/account](https://airtable.com/account)
   - Scroll down to **"API"** section
   - Click **"Generate new token"**
   - Copy the **API Key** (starts with `key`)

### **Step 6: Set Environment Variables**

```bash
export AIRTABLE_API_KEY="your_api_key_here"
export AIRTABLE_BASE_ID="your_base_id_here"
```
