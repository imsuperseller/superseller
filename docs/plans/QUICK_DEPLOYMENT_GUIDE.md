## 📞 TROUBLESHOOTING

**If content doesn't show:**
- Make sure you copied the ENTIRE file
- Check that Custom Code element is not hidden
- Refresh preview

**If animations don't work:**
- Check browser console for GSAP loading errors
- Make sure internet connection is active (GSAP loads from CDN)

**If styles are broken:**
- Make sure `<style>` tags were included in paste
- Check for conflicting CSS from other elements
