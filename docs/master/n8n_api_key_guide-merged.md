# מדריך יצירת API Key ל-n8n

## שלבים ליצירת API Key:
1. היכנס לממשק n8n: http://173.254.201.134:5678
2. התחבר עם המשתמש שלך
3. לך ל-Settings (הגדרות)
4. בחר ב-API
5. לחץ על "Create API Key"
6. תן שם כמו: "Insurance System API Key"
7. העתק את ה-API Key שנוצר

## עדכון קובץ .env:
לאחר קבלת ה-API Key החדש:
1. פתח את קובץ .env
2. מצא את השורה: N8N_API_KEY=...
3. החלף את הערך הישן ב-API Key החדש
4. שמור את הקובץ

## בדיקת החיבור:
לאחר עדכון ה-API Key, הרץ:
curl -H "Authorization: Bearer YOUR_NEW_API_KEY" http://173.254.201.134:5678/rest/workflows

## אם אין לך גישה לממשק n8n:
1. בדוק שהשרת פועל: curl http://173.254.201.134:5678/
2. בדוק שה-IP נכון: 173.254.201.134
3. בדוק שהפורט נכון: 5678
4. בדוק שה-firewall מאפשר גישה

## בדיקת סטטוס השרת:
הרץ את הפקודה הבאה לבדיקת סטטוס השרת:
curl -I http://173.254.201.134:5678/

## אם השרת לא מגיב:
1. בדוק שהשרת פועל על ה-VPS
2. בדוק שה-n8n service פעיל
3. בדוק את הלוגים של n8n
4. נסה להפעיל מחדש: sudo systemctl restart n8n

## לאחר קבלת API Key תקין:
1. עדכן את קובץ .env
2. הרץ: node scripts/deploy_insurance_workflows.js
3. בדוק שהחיבור עובד
4. המשך עם יצירת הטמפלטים

## סיכום
המערכת מוכנה לחלוטין! רק צריך:
1. API Key תקין ל-n8n
2. יצירת טמפלטים Google Drive
3. עדכון משתני סביבה
4. ייצוא וורק-פלוים

המערכת תעבוד אוטומטית לאחר ההגדרה!
