# Experiments Directory

This directory contains experimental code, prototypes, and variations that are not part of the canonical codebase.

## Directory Structure

### `/variations`
- **Purpose**: UI/UX design variations and prototypes
- **Status**: Experimental
- **Last Updated**: August 2025
- **Description**: Various design iterations and UI experiments

### `/variations3`
- **Purpose**: Additional design variations and prototypes
- **Status**: Experimental
- **Last Updated**: August 2025
- **Description**: More design iterations and UI experiments

### `/infinite-ui-cursor`
- **Purpose**: Cursor-specific UI experiments and prototypes
- **Status**: Experimental
- **Last Updated**: August 2025
- **Description**: UI experiments specifically for Cursor IDE integration

## Migration Notes

These directories were moved from the root level as part of the Zero-Dupes architecture implementation. They contain experimental code that should not be considered part of the production codebase.

## Guidelines

1. **Experimental Code**: All code in this directory is experimental and not guaranteed to work
2. **No Dependencies**: Experiments should not create dependencies on canonical packages
3. **Documentation**: Each experiment should include a README explaining its purpose
4. **Cleanup**: Experiments should be cleaned up or moved to canonical code when proven

## Future Plans

- Consolidate similar experiments
- Move successful experiments to canonical codebase
- Archive failed experiments
- Maintain clear separation between experimental and production code
