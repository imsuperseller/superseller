# 🚀 RENSTO WEBSITE DEVELOPMENT BRIEF

**Date**: January 16, 2025  
**Project**: Rensto Universal Automation Platform Website  
**Status**: 🎯 **READY FOR DEVELOPMENT**

---

## 📋 **PROJECT OVERVIEW**

### **Business Model Transformation**
Rensto is transforming from a traditional business operations platform to a **Universal Micro-SaaS Automation Platform**. The new website must reflect this transformation completely.

**Key Changes:**
- **From**: Custom implementations for specific customers (Tax4Us, Shelly, Ortal)
- **To**: Universal subscription platform that works for ANY industry/niche
- **Revenue Model**: Subscription-based with usage billing
- **Target**: Self-service customers who can set up their own systems

---

## 🎯 **WEBSITE ARCHITECTURE**

### **Public Website Structure**
```
/ (Home) - Universal subscription platform landing
├── /pricing (Subscription Plans)

├── /contact (Sales & Support)
├── /legal/privacy (Privacy Policy)

```

### **Customer App Structure**
```
/app (Customer Portal - Protected)
├── /dashboard (Usage & Analytics)
├── /templates (Universal Templates)
├── /agents (AI Agent Management)
├── /billing (Subscription Management)
├── /settings (Account Configuration)

```

### **Admin Dashboard Structure**
```
/admin (Internal Management - Protected)
├── /customers (Customer Management)
├── /revenue (Revenue Analytics)
├── /system (System Monitoring)
└── /workflows (Workflow Management)
```

## 🛠️ **TECHNICAL REQUIREMENTS**

### **Technology Stack**
- **Framework**: Next.js 14+ with App Router
- **Styling**: Tailwind CSS + shadcn/ui + tweakcn
- **Components**: React components with TypeScript
- **Authentication**: NextAuth.js for protected routes
- **Database**: Integration with existing Airtable bases
- **Payments**: Stripe integration for subscriptions

### **Performance Requirements**
- **Page Load Speed**: < 2 seconds
- **Mobile Responsiveness**: 100% optimized
- **SEO**: Fully optimized for search engines
- **Accessibility**: WCAG 2.1 AA compliant

### **Integration Requirements**
- **Existing Admin Dashboard**: Integrate with current admin system
- **Customer Portal**: Connect to existing customer management
- **Airtable**: Sync with existing customer data
- **n8n Workflows**: Connect to automation system

## 📄 **PAGE SPECIFICATIONS**

### **1. Home Page (`/`)**
**Purpose**: Convert visitors to trial signups
**Key Elements**:
- Hero: "Universal Automation Platform for Any Industry"
- Value Props: Universal, Self-Service, Scalable
- Social Proof: Customer logos and testimonials
- CTA: "Start Free Trial" / "View Pricing"
- Features: Universal templates, AI agents, real-time analytics

### **2. Pricing Page (`/pricing`)**
**Purpose**: Convert visitors to paid subscriptions
**Plans**:
- **Starter**: $97/month - Basic features, 100 agent interactions
- **Professional**: $197/month - Advanced features, 500 agent interactions
- **Enterprise**: $497/month - White-label, unlimited interactions
**Features**: Usage-based billing, no setup fees, cancel anytime

### **3. Features Page (`/features`)**
**Purpose**: Educate visitors on platform capabilities
**Sections**:
- Universal Templates (works for any industry)
- AI Agent Management (self-service setup)
- Real-time Analytics (usage tracking)
- Self-Service Portal (customer independence)
- White-label Options (enterprise features)

### **4. About Page (`/about`)**
**Purpose**: Build trust and credibility
**Content**:
- Company mission and vision
- Team information
- Customer success stories
- Industry expertise
- Technology leadership

### **5. Contact Page (`/contact`)**
**Purpose**: Generate leads and provide support
**Elements**:
- Contact form
- Support options
- Sales contact information
- FAQ section
- Demo request form
