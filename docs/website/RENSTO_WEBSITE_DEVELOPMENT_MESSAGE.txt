## 🎯 **PROJECT OVERVIEW**

Hi there! I need you to build a completely new Rensto website that reflects our **major business model transformation**. We're moving from a traditional business operations platform to a **Universal Micro-SaaS Automation Platform**.

### **Key Transformation**
- **From**: Custom implementations for specific customers (Tax4Us, Shelly, Ortal)
- **To**: Universal subscription platform that works for ANY industry/niche
- **Revenue Model**: Subscription-based with usage billing
- **Target**: Self-service customers who can set up their own systems

---

### **Public Website** (No authentication required)
1. **Home Page** (`/`) - Universal subscription platform landing

## 🎨 **DESIGN REQUIREMENTS**

### **Brand Guidelines**
- **Colors**: #fe3d51 (primary), #bf5700 (secondary), #1eaef7 (accent), #5ffbfd (highlight) on #110d28 (background)
- **Typography**: Inter font family, modern and clean
- **Style**: Bold, confident, tech-forward
- **Logo**: Rensto logo must be included in all design outputs

### **Design System**
- Use **shadcn/ui** components as base
- Implement **tweakcn** for enhanced components
- Ensure **100% mobile responsiveness**
- Follow **modern SaaS design patterns**

---

## 🛠️ **TECHNICAL REQUIREMENTS**

### **Technology Stack**
- **Framework**: Next.js 14+ with App Router
- **Styling**: Tailwind CSS + shadcn/ui + tweakcn
- **Components**: React components with TypeScript
- **Authentication**: NextAuth.js for protected routes
- **Database**: Integration with existing Airtable bases
- **Payments**: Stripe integration for subscriptions

### **Performance Requirements**
- **Page Load Speed**: < 2 seconds
- **Mobile Responsiveness**: 100% optimized
- **SEO**: Fully optimized for search engines
- **Accessibility**: WCAG 2.1 AA compliant
