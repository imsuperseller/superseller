## 🔧 TROUBLESHOOTING

### **If content doesn't show:**
1. Make sure you copied the ENTIRE file (including `<style>` tags)
2. Check that Custom Code embed is enabled (not disabled)
3. Try refreshing the preview

### **If animations don't work:**
1. Check browser console for errors
2. Make sure GSAP CDN is loading (included in each file)
3. Try clearing cache and reloading

### **If layout is broken:**
1. Make sure no other CSS is conflicting
2. Check that the embed is at full width
3. Try viewing in incognito/private mode
