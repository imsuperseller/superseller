## 🎯 What's New in 2.1.4 (vs 2.1.0)

### **Bug Fixes & Improvements**

1. **OAuth2 Cloud Hook Integration** (2.1.3)
   - Fixed issues with OAuth2 authentication in cloud hook integrations
   - Better reliability for OAuth-based workflows

2. **Expression Handling** (2.1.3)
   - Improved expression evaluation and handling
   - Better error messages for expression issues

3. **Markdown Rendering** (2.1.3)
   - Improved markdown rendering in the UI
   - Better display of documentation and descriptions

4. **Ollama Models Support** (2.1.2)
   - Fixed support for Ollama models in Chat hub
   - Better compatibility with local AI models

5. **Community Packages** (2.1.2)
   - Fixed verified community packages reinstall issue
   - Better handling of community node installations

6. **Workflow Statistics** (2.1.2)
   - Skip saving workflow statistics on 'chat' executions
   - Improved performance for chat-based workflows

7. **File Access Security** (2.1.2)
   - Limit access to files based on regex pattern
   - Enhanced security for file operations

8. **Form Node** (2.1.2)
   - Restored executions status check for waiting forms
   - Better handling of form submissions

9. **AI Builder** (2.1.1)
   - Connection validation respects node version and type
   - Better validation for AI node connections

10. **Queue Mode** (2.1.1)
    - Fixed error running evaluations in queue mode
    - Better reliability for queue-based executions

11. **Hooks Calls** (2.1.1)
    - Don't encrypt state for hooks calls
    - Improved performance for webhook executions

12. **Workflow Activation Events** (2.1.1)
    - Fixed workflow nodes not updated correctly on activation/deactivation
    - Better UI synchronization

13. **Chat Hub** (2.1.1)
    - Select default Chat hub model from models allowed by settings
    - Better model selection in chat interface

14. **Git Node** (2.1.1)
    - Only support specified git config keys
    - Enhanced security for Git operations
