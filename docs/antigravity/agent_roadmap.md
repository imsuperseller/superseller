# Rensto Domain Agent Roadmap: "The Terry Squad"

We currently have **Terry** (Infra Monitoring). To transform Rensto into a fully AI-operated automation platform, we need domain-specific "Operatives" that bridge your tech stack with business outcomes.

## 1. 💹 Finny (The Profit Guardian)
**Goal**: Real-time financial intelligence and proactive churn management.
- **Tools**: Stripe API, QuickBooks n8n nodes, Firestore `payments` collection.
- **Capabilities**:
    - "Finny, who is our highest LTV customer but has low engagement?"
    - "Generate a P&L for Tax4Us from Dec 2025."
    - "Alert me if any subscription over $500 fails twice."

## 2. 🎬 Conti (The Growth Engine)
**Goal**: Autonomous content distribution and engagement analysis.
- **Tools**: YouTube API, Facebook/IG/TikTok APIs, n8n Content AI, WordPress.
- **Capabilities**:
    - "Conti, take the podcast transcript from n8n and schedule 5 'modkit' clips to TikTok."
    - "How is the 'Sell Outcomes' campaign performing across all social channels?"
    - "Repurpose our latest blog post into a 10-post LinkedIn series."

## 3. ⚙️ Ops-Szen (The Specialized Manager)
**Goal**: Managing complex, customer-specific automation clusters (e.g., Szender/GoLogin).
- **Tools**: GoLogin API, WAHA Pro, Custom szender n8n workflows.
- **Capabilities**:
    - "Ops-Szen, check the status of all 50 GoLogin profiles for the Szender project."
    - "Rotate IPs for the scraping cluster if the success rate drops below 80%."
    - "Provision a new WhatsApp routing instance for a Szender sub-client."

## 4. 🌩️ Cloud-Terry (The Scaling Operative)
**Goal**: Managing the Vercel/Cloudflare/Racknerd backbone.
- **Tools**: Vercel API, Cloudflare Workers, Firebase Admin SDK.
- **Capabilities**:
    - "Terry, we're expecting a traffic spike. Increase Cloudflare rate limits for the API."
    - "Deploy a new staging environment for the Marketplace update."
    - "Check Firestore index health and optimize slow queries."
