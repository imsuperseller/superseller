## 🎯 **OVERVIEW**

**Goal**: Create a stylized, illustrated avatar that serves as:
1. **Video Presenter** - For all HeyGen videos
2. **Brand Mascot** - For marketing materials, social media, website
3. **Consistent Brand Identity** - Recognizable across all touchpoints

**Why Stylized?**
- ✅ More memorable than realistic avatars
- ✅ Works as both video presenter AND mascot
- ✅ Easier to maintain brand consistency
- ✅ More versatile across different contexts
- ✅ Timeless (won't age like realistic photos)
- ✅ Can be animated/illustrated for other uses

**Base Avatar**: Stylized illustrated character (to be created)

**Brand Colors**: 
- Primary Red: `#fe3d51`
- Secondary Orange: `#bf5700`
- Accent Blue: `#1eaef7`
- Accent Cyan: `#5ffbfd`
- Dark Background: `#110d28`

## 🎨 **AVATAR DESIGN SPECIFICATIONS**

### **Character Style**
- **Type**: Stylized illustrated character (not realistic)
- **Style**: Modern, professional, approachable
- **Aesthetic**: Tech-forward but friendly, clean lines, vibrant colors
- **Inspiration**: Blend of professional consultant + tech innovator + friendly guide

### **Physical Characteristics**
- **Age**: 30-40 (professional but approachable)
- **Build**: Average, professional appearance
- **Facial Features**: Friendly, confident expression, approachable smile
- **Hair**: Modern, professional style (can vary in different looks)
- **Overall**: Professional but not intimidating, tech-savvy but accessible

### **Brand Alignment**
- **Colors**: Incorporate SuperSeller AI brand colors (#fe3d51, #1eaef7, #bf5700, #5ffbfd)
- **Style**: Modern, clean, tech-forward
- **Personality**: Knowledgeable, helpful, approachable, innovative
- **Mood**: Professional but friendly, authoritative but accessible

## 🛠️ **CREATION METHODS**

### **Method 1: HeyGen Generative Avatar** (Recommended)

**Step 1: Create Base Avatar**
1. Go to HeyGen → Avatars → "Generative Avatar"
2. Enter detailed prompt (see prompts below)
3. Generate multiple variations
4. Select best base avatar
5. Save as "SuperSeller AI Mascot Base"

**Step 2: Create Different Looks**
1. Use "Avatar Looks" feature
2. Generate different outfits/contexts
3. Save each as named look (e.g., "Professional Consultant Look")

### **Method 2: Midjourney → HeyGen** (Alternative)

**Step 1: Generate Reference Images in Midjourney**
1. Use Midjourney prompts below
2. Generate 5-10 variations
3. Select best 2-3 designs
4. Upscale selected images

**Step 2: Upload to HeyGen**
1. Use HeyGen's image-to-avatar feature (if available)
2. Or use as reference for Generative Avatar prompts
3. Create avatar based on reference

### **Method 3: Professional Designer** (If needed)

If AI generation doesn't meet requirements:
1. Hire illustrator to create character design
2. Export as high-res images
3. Upload to HeyGen as custom avatar
4. Create looks from base design
