## 📚 **HEYGEN AVATAR LOOKS - BEST PRACTICES**

### **What Are Avatar Looks?**
HeyGen's Avatar Looks feature allows you to:
- Change avatar's **outfit** (business casual, formal, casual)
- Change **background** (office, studio, modern workspace)
- Adjust **lighting** and **mood** (professional, friendly, authoritative)
- Maintain **same face/voice** (your @shai avatar)
- **No re-recording needed** - apply looks instantly

### **Best Practices**:
1. **Consistency**: Use same look per video series (e.g., all Tailored Solutions videos use "Professional Consultant" look)
2. **Audience Match**: Match look to target audience (formal for enterprise, casual for SMB)
3. **Brand Alignment**: Keep colors aligned with SuperSeller AI brand (#fe3d51, #1eaef7, #bf5700, #5ffbfd)
4. **Context Appropriate**: Match look to video content (consultation = professional, demo = approachable)

## 🛠️ **HOW TO CREATE AVATAR LOOKS IN HEYGEN**

### **Method 1: Using HeyGen Avatar Looks Feature** (Recommended)

1. **Open HeyGen** → Go to "Avatars" section
2. **Select Your Avatar** (@shai)
3. **Click "Avatar Looks"** or "Create Look"
4. **Enter Look Prompt** (use prompts from above)
5. **Customize**:
   - Outfit selection
   - Background selection
   - Lighting adjustment
   - Save as named look (e.g., "Professional Consultant")
6. **Apply to Video**: When creating video, select your avatar → Choose saved look

### **Method 2: Using Generative Avatar** (Alternative)

If you want to create completely new avatar appearances:

1. **Open HeyGen** → "Avatars" → "Generative Avatar"
2. **Enter Detailed Prompt** (use prompts from above, add "based on @shai appearance")
3. **Generate** → Review → Save as new look
4. **Use in Videos**: Select generated avatar for specific video types

### **Method 3: Using Midjourney for Reference Images** (Optional)

If you want visual references before creating looks in HeyGen:

1. **Generate Reference Image** in Midjourney using prompts below
2. **Upload to HeyGen** as reference
3. **Create Look** based on reference image

## 📝 **STEP-BY-STEP: CREATING YOUR FIRST LOOK**

### **Example: Professional Consultant Look**

1. **Open HeyGen** → Log in
2. **Navigate**: Avatars → Select @shai → "Avatar Looks" or "Create Look"
3. **Enter Look Name**: "Professional Consultant"
4. **Enter Description/Prompt**:
   ```
   Professional business consultant, dark navy suit, white dress shirt, modern office background with subtle SuperSeller AI brand colors, professional studio lighting, warm tone, authoritative but approachable, trustworthy expert appearance
   ```
5. **Customize Settings**:
   - **Outfit**: Select "Business Suit" → Dark navy
   - **Background**: Select "Modern Office" → Add SuperSeller AI logo visible
   - **Lighting**: "Professional Studio" → Warm tone
   - **Mood**: "Authoritative but Approachable"
6. **Preview** → Adjust if needed
7. **Save Look** as "Professional Consultant"
8. **Test**: Create test video → Select @shai → Apply "Professional Consultant" look
9. **Review**: Does it match the video type? Adjust if needed
