## 🛠️ **HOW TO USE HEYGEN VIDEO AGENT**

### **Step 1: Access Video Agent**
1. Go to HeyGen.com
2. Click "Create Video" → "Video Agent"
3. Select "Generate from Prompt"

### **Step 2: Paste Optimized Prompt**
1. Copy entire prompt from above (includes script + visual requirements)
2. Paste into HeyGen Video Agent
3. Select brand (SuperSeller AI - Brand Shuffle will auto-apply colors/fonts)

### **Step 3: Generate Video**
1. Click "Generate Video"
2. Wait 2-5 minutes
3. Review generated video

### **Step 4: Refine (if needed)**
1. If visuals need adjustment, add specific notes
2. Regenerate or manually adjust elements
3. Export 1080p

### **Step 5: Integrate**
1. Upload to YouTube (unlisted)
2. Get video ID
3. Integrate into page (see integration guide)
4. Deploy to production
