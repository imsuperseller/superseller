## 🎯 **PRIMARY PROMPT** (Use This First)

**Duration**: 10 seconds  
**Format**: Vertical (1080x1920, 30fps)

```
Generate a 10-second vertical format video (1080x1920, 30fps) featuring a stylized illustrated professional business consultant character speaking directly to the camera. The character is a 30-40 year old professional with a friendly, confident expression and approachable smile. They have a modern professional appearance with clean illustrated style, Pixar-quality animation. The character wears a modern professional outfit incorporating vibrant brand colors: primary red (#fe3d51), accent blue (#1eaef7), secondary orange (#bf5700), and accent cyan (#5ffbfd). The character's hair is styled in a modern professional manner. Their facial features are friendly and confident with an approachable smile, professional but not intimidating appearance. The character stands or sits in a clean modern workspace background with subtle tech-forward elements visible. The lighting is bright and professional, evenly illuminating the character's face. The character speaks directly to the camera with natural hand gestures, maintaining eye contact throughout. The video maintains a consistent illustrated style throughout, with smooth character movement and natural speaking animation. The character's appearance is consistent and recognizable, suitable for establishing a brand mascot cameo character.
```

## 🔧 **TROUBLESHOOTING**

### **If Character Appearance Varies**:
- Add more specific details about facial features
- Specify "consistent appearance throughout" multiple times
- Use "same character" or "identical character" in prompt

### **If Colors Don't Appear**:
- Be more specific about where colors appear (outfit, accessories, background elements)
- Use color names AND hex codes
- Mention "vibrant brand colors" multiple times

### **If Animation is Not Smooth**:
- Add "smooth animation" and "natural movement" to prompt
- Specify "Pixar-quality" or "professional animation"
- Request "consistent frame-to-frame animation"

### **If Cameo Creation Fails**:
- Ensure character appears clearly in every frame
- Character should be the main focus (not background)
- Character should face camera directly
- Video should be 10-15 seconds (Sora's standard)
