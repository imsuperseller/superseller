## 📚 **HEYGEN BEST PRACTICES (From Research)**

### **For Look Prompts**:
1. ✅ **Be Specific**: Include attire, hairstyle, accessories, background
2. ✅ **Structured Format**: Image Type + Main Subject + Background Scene + Composition Style
3. ✅ **Include Context**: Setting and mood guide AI generation
4. ✅ **Use Descriptive Language**: Adjectives convey emotion and style

### **For Motion Prompts**:
1. ✅ **Clear and Direct**: Specify body part + action + emotion
2. ✅ **One Gesture at a Time**: Don't combine multiple actions
3. ✅ **Natural Timing**: HeyGen handles pacing automatically
4. ✅ **Include Facial Expressions**: "smiles gently" or "looks surprised"
