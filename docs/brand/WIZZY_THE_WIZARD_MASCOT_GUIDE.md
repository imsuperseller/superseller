### **🎯 Midjourney v7 Optimization Guide**

**Why v7 is Perfect for Wizzy**:

Midjourney v7 introduces **Omni-Reference** (`--oref`) - a game-changer for character consistency. This means you can:
1. Generate Base Wizzy first
2. Use that image as a reference for ALL other Wizzy variations
3. Maintain perfect character consistency across poses, scenes, and expressions

**Step-by-Step Workflow**:

**Phase 1: Create Base Wizzy**
```
1. Generate Base Wizzy using the prompt above
2. Save the best result as "wizzy-base.png"
3. Upload to Midjourney for reference URL
```

**Phase 2: Generate Consistent Variations**
Once you have Base Wizzy, use **Omni-Reference** for all other poses:

```
[Same character description] in [new pose/action] --oref [wizzy-base-url] --ow 500 --stylize 500 --ar 1:1 --v 7
```

**Example - Casting Wizzy with Omni-Reference**:
```
Wizzy the wizard casting a spell, wand extended with cyan energy flowing, connecting tool icons, excited expression --oref https://s.mj.run/[your-wizzy-base-url] --ow 500 --stylize 500 --ar 1:1 --v 7
```

**Parameter Breakdown**:
- `--oref [URL]`: Reference image for character consistency
- `--ow 500`: Omni-Weight (400-600 recommended for characters; higher = more exact replica)
- `--stylize 500`: Balanced artistic interpretation (not too literal, not too creative)
- `--v 7`: Use Midjourney version 7 (latest)

**v7 Advantages for Wizzy**:
✅ **Better faces**: Fewer anomalies, more coherent expressions  
✅ **Consistent character**: Omni-Reference maintains Wizzy's look across all images  
✅ **Natural realism**: Softer tones, better anatomy for friendly wizard aesthetic  
✅ **Draft Mode**: 10x faster previews for rapid iteration  
✅ **Improved prompt understanding**: Natural language works better than keyword lists  

**Pro Tips**:
- Use **natural conversational language** (v7 understands context better)
- **Describe the scene**, not just keywords
- Use `--stylize 500` as baseline (adjust 300-700 based on results)
- For clothing/outfit accuracy, try `--ow 400-600`
- Generate Base Wizzy in highest quality, then reference it for all variations
