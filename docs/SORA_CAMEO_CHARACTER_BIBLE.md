# Sora Cameo Character Bible

**Purpose**: Reusable character prompts for Sora's cameo creation interface. Each character becomes a persistent @handle that can appear across multiple videos. Paste the prompt verbatim into Sora's cameo creation prompt field.

**Last Updated**: March 10, 2026
**Author**: Generated from Brand Identity Bible + Customer Avatar ICP

---

## How to Use This Document

1. Go to Sora → Cameo → Create New Character
2. Paste the prompt exactly as written under "Sora Cameo Creation Prompt"
3. Set the @handle to the name specified
4. Generate and save — Sora will lock in the face/look for reuse
5. Reference the character in video prompts as `@handle` doing the described use cases

## SS-01: "The Tech Expert" — @superseller_tech

**Handle**: `@superseller_tech`
**Business**: SuperSeller AI
**Character Role**: Represents AI capability — the "smartest person in the room" who makes it feel simple

### Sora Cameo Creation Prompt

```
A confident male software engineer in his mid-30s, Israeli-American appearance, dark olive skin, short black wavy hair neatly styled, clean-shaven with subtle stubble. He wears a fitted navy blue crewneck with a subtle orange geometric logo on the chest, dark jeans. No tie, no suit — smart casual tech professional. He is seated at a modern dark-toned workstation with dual monitors showing glowing teal and orange dashboard interfaces in the background. Warm studio key light from the left, subtle blue ambient fill from monitor screens. His expression is engaged and confident — like someone explaining something he genuinely loves, not reading a script. Slight forward lean. Brown-hazel eyes, straight posture, slight smile. Mediterranean features — think Tel Aviv tech scene but based in Dallas.
```

### Use Cases

- Explainer videos: "Here's how the AI Crew works in 60 seconds"
- Feature announcements: new product launches
- "Behind the AI" educational content
- Side-by-side comparison: manual work vs AI automation
- Social media ads targeting tech-adjacent small business owners

### Voice Recommendation

- **ElevenLabs voice type**: Professional male, American accent with very slight warm undertone, confident pace — not robotic. Think "smart friend explaining, not salesperson selling."
- **Clone target**: Record 5 minutes of calm explainer content at 120 WPM for training
- **Voice ID style**: `eleven_multilingual_v2`, stability 0.65, similarity 0.80
