# 📖 Jargon Dictionary

> **Source of Truth for all Rensto-specific terminology.**

---

| Term | Definition |
| :--- | :--- |
| **Pillar** | One of the 4 main deliverables (Lead Machine, Autonomous Secretary, Knowledge Engine, Content Engine). |
| **Engine** | A product (e.g., Lead Machine) that drives a specific business outcome. Implies power and automation. |
| **Modkit** | A standalone, modular feature that can be added to a workflow to enhance capabilities. |
| **Blueprint** | The raw JSON/workflow file sold in the marketplace. A "download-only" tier. |
| **Service Instance** | A specific deployment of a workflow for a client. Tracked in Firestore `service_instances` collection. |
| **Automation Audit** | A $497 diagnostic service that identifies automation opportunities before recommending solutions. |
| **Core 7** | The seven flagship products with verified schemas and active Stripe checkout. |
| **Fulfillment Queue** | The admin dashboard view (`/control/fulfillment`) for managing pending service orders. |
| **LightRAG** | The Graph-RAG engine used for the Knowledge Engine pillar, hosted on the Rensto VPS. |
