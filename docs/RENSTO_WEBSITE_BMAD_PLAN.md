## **🎯 CORE PRINCIPLES (Deiss/DigitalMarketer)**

### **1. Customer Value Journey (CVJ) Mapping**
Every page and CTA must intentionally move visitors through:
- **Aware** → **Engage** → **Subscribe** → **Convert** → **Excite** → **Ascend** → **Advocate** → **Promote**

### **2. Offer Architecture (CVO)**
- **Lead Magnet**: Free lead generation audit/tool
- **Tripwire**: $97 Quick Lead Setup
- **Core Offer**: Monthly lead generation service ($497-$997)
- **Profit Maximizer**: White-label reseller program
- **Return Path**: Upsell to premium features

### **3. Homepage as System Hub**
Route traffic to the right next step for each intent segment using proven section-by-section formula.

## **🎨 COPYWRITING RULES (Deiss-Style)**

### **1. Lead with Fit**
- First lines specify who it's for
- Self-qualify the audience

### **2. Problem → Why it Persists → Mechanism → Outcome → CTA**
- Keep this arc in every section
- Not just the page

### **3. Proof Early and Often**
- Each scroll segment pairs claim with evidence
- Testimonials, case studies, numbers

### **4. One Goal Per Page**
- One primary CTA per viewport
- Secondary CTA is visually subordinate

### **5. Sequence, Not Slogans**
- Every CTA maps to CVJ stage
- Don't jump stages
