## **🚀 DEPLOYMENT OPTIONS**

### **Option 1: Vercel (Recommended)**
```bash
cd web/rensto-site
npm i -g vercel
vercel --prod
```

### **Option 2: Docker**
```bash
cd web/rensto-site
docker build -t rensto-business-system .
docker run -p 3000:3000 rensto-business-system
```

### **Option 3: Manual Server**
```bash
cd web/rensto-site
npm ci --production
npm start
```
