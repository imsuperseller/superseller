Getting Started with Rensto

Learn how to set up and use Rensto automation platform to transform your business operations.

📅 Last updated: March 15, 2025 ⏱️ 15 min read 👤 Rensto Team

Introduction

Welcome to Rensto, the universal automation platform that helps businesses streamline their operations with AI-powered workflows. This guide will walk you through the basics of getting started with our platform.

What is Rensto?

Rensto is a comprehensive automation platform that offers four main service types:

  • Marketplace: Pre-built n8n templates and professional installation
  • Custom Solutions: Voice AI consultation and tailored automation development
  • Subscriptions: Enhanced lead generation with CRM integration
  • Ready Solutions: Industry-specific automation packages

Quick Start

To get started with Rensto, follow these simple steps:

💡 Pro Tip

Start with our Marketplace templates if you're new to automation. They're designed for beginners and come with professional installation.

Step 1: Choose Your Service Type

Select the service type that best fits your business needs:

  • For DIY users: Marketplace templates
  • For complex needs: Custom Solutions
  • For lead generation: Subscriptions
  • For industry-specific needs: Ready Solutions

Step 2: Set Up Your Account

Create your Rensto account and configure your preferences:

# Install Rensto CLI npm install -g @rensto/cli # Login to your account rensto login # Configure your workspace rensto config --workspace=your-workspace-id

Step 3: Create Your First Workflow

Follow our step-by-step tutorial to create your first automation workflow:

  1. Access the Rensto dashboard
  2. Choose a template or start from scratch
  3. Configure your workflow nodes
  4. Test and deploy your automation

Next Steps

API Reference

Complete API documentation for integrating with Rensto

GET
/api/v1/workflows

Retrieve all workflows for your account with optional filtering and pagination.

  • limit integer
  • offset integer
  • status string
POST
/api/v1/workflows

Create a new workflow with the specified configuration and settings.

  • name string
  • nodes array
  • connections object
PUT
/api/v1/workflows/{id}

Update an existing workflow with new configuration or settings.

  • id string
  • name string
  • active boolean
DELETE
/api/v1/workflows/{id}

Delete a workflow and all associated data. This action cannot be undone.

  • id string
