Create `apps/worker/src/services/voice-transcription.ts` with:

1. **initVoiceTranscriptionTable()** — Raw SQL CREATE TABLE IF NOT EXISTS:
   ```sql
   CREATE TABLE IF NOT EXISTS voice_transcriptions (
     id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
     tenant_id UUID REFERENCES "Tenant"(id),
     r2_key TEXT NOT NULL,
     transcription TEXT NOT NULL,
     duration_seconds FLOAT,
     language_detected TEXT,
     wa_message_id TEXT,
     wa_chat_id TEXT,
     created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
   );
   CREATE INDEX IF NOT EXISTS idx_voice_transcriptions_tenant ON voice_transcriptions(tenant_id);
   CREATE INDEX IF NOT EXISTS idx_voice_transcriptions_created ON voice_transcriptions(created_at);
   ```

2. **transcribeVoiceNote(params)** — Main function accepting:
   ```typescript
   interface TranscribeParams {
     audioBuffer: Buffer;
     contentType: string;       // "audio/ogg", "audio/mpeg", "video/mp4", etc.
     tenantId: string;
     chatId: string;
     messageId: string;
     durationSeconds?: number;  // From WAHA if available
   }
   ```
   Returns: `{ text: string; language: string; duration: number } | null`

   Implementation:
   - If contentType starts with "video/", call extractAudioFromVideo() first to get audio buffer
   - Upload original audio to R2: key = `voice-notes/{tenantId}/{Date.now()}-{messageId}.{ext}` using mimeToExt helper (copy from ep-asset-ingestion.ts pattern)
   - Call OpenAI Whisper API: POST https://api.openai.com/v1/audio/transcriptions with FormData containing file (as Blob with correct mime), model="whisper-1", response_format="verbose_json" (to get language + duration)
   - Auth: Bearer token from `process.env.OPENAI_API_KEY`
   - Retry logic: 3 attempts with exponential backoff (1s, 2s, 4s) on 5xx or network errors. No retry on 4xx.
   - If response has empty text or Whisper returns error, return null
   - Extract `text`, `language`, `duration` from verbose_json response
   - Insert into voice_transcriptions table
   - Call trackExpense with service='openai', operation='whisper_transcription', estimatedCost = (duration / 60) * 0.006, metadata = { tenantId, language, durationSeconds: duration }
   - Return { text, language, duration }

3. **extractAudioFromVideo(videoBuffer)** — Returns audio Buffer:
   - Write video buffer to temp file (os.tmpdir)
   - Run `ffmpeg -i {input} -vn -acodec libopus -f ogg {output}` via child_process.execSync
   - Read output file into buffer, clean up temp files
   - Return audio buffer (or throw on FFmpeg failure)

4. **Add COST_RATES entry** in expense-tracker.ts: `openai: { whisper_transcription: 0.006 }` (per minute, but actual cost is computed dynamically in transcribeVoiceNote)

5. **Duration validation**: If durationSeconds > 300 (5 minutes), return null immediately (per user decision on max duration).
