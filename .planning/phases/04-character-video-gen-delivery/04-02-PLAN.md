2. Create `CharacterRevealComposition.tsx` adapting patterns from HairShowreelComposition:

**Structure:**
- IntroCard (3.5s): Dark background, character name in Playfair Display font, tagline, logo if available, accent color accents, fade-in animation with spring scale
- 5 Scene clips (5s each): Each rendered via `<OffthreadVideo>`, with:
  - Scene label overlay at bottom (white text on semi-transparent dark bar, accent color underline)
  - Subtle film grain via `<FilmGrain />`
  - Vignette via `<Vignette />`
- OutroCard (3.5s): "Your AI Character" title, character name, business name, accent-colored divider, fade-out
- Transitions: Use `<TransitionSeries>` with `linearTiming` — alternate between fade/slide/wipe transitions (0.6s each), matching HairShowreel pattern

**Timing:**
- INTRO_DURATION = sec(3.5)
- SCENE_DURATION = sec(5) per clip
- TRANSITION_FRAMES = sec(0.6)
- OUTRO_DURATION = sec(3.5)
- Total: ~32s for 5 scenes

**Video specs:**
- 1920x1080 (16:9 landscape) — fits WhatsApp's ~16MB limit at reasonable quality
- 30fps (from existing FPS constant)
- No audio/music (explicitly out of scope per REQUIREMENTS.md)

**Style:** Dark luxury aesthetic with warm accent colors. Use `COLOR_GRADE_STYLE` filter pattern from HairShowreel (saturate 1.1, contrast 1.05). Warm overlay for cinematic feel.

**Scene label overlay:** Position at bottom 15% of frame, semi-transparent black bar (rgba(0,0,0,0.6)), white text in FONT_FAMILY, accent-colored 2px underline. Fade in 0.5s after scene starts.

**Handle missing clips:** If `sceneClips` has fewer than 5 entries, only render available scenes. Skip missing indices gracefully (filter nulls/empty strings).
