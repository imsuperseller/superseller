## 🎯 **KEY PRINCIPLES**

### **🤖 AI Agent Usage Guidelines**
- **DON'T USE** for simple if/else or switch statements
- **USE** for complex reasoning and infinite possibilities
- **Validation**: Automatic detection of inappropriate usage
- **Integration**: Built into workflow planning and validation

### **⏰ Proactive Automation**
- **AVOID**: Manual triggers and human dependency
- **USE**: Schedulers and automatic triggers
- **Patterns**: Daily, hourly, and event-based automation
- **Benefits**: Consistent execution and scalability

### **📉 Complexity Reduction**
- **Target**: < 25 nodes per workflow
- **Scoring**: Weighted factors (nodes, branches, integrations, errors, transformations)
- **Strategies**: Break down workflows, use subworkflows, simplify conditionals
- **Monitoring**: Real-time complexity tracking

### **👤 Human-in-the-Loop**
- **Critical Decisions**: Content publishing, financial transactions, customer communications
- **Templates**: Content generation, data analysis, customer communication workflows
- **Approval Mechanisms**: Email, dashboard, Slack integration
- **Error Handling**: Automatic retry with human escalation

### **💰 ROI-Focused Development**
- **Minimum ROI**: 2.0x time savings
- **Payback Period**: Maximum 30 days
- **Calculation**: (Time Saved × Frequency) / Development Time
- **Validation**: Pre-development ROI calculation required
